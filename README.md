# HW5 – UFO Sightings Visualizations
**IS 457 · Data Visualization**

Two Altair visualizations built on the [UFO Scrubbed Geocoded Time Standardized](https://github.com/UIUC-iSchool-DataViz/is445_data/raw/main/ufo-scrubbed-geocoded-time-standardized-00.csv) dataset.

- **Plot 1:** Heatmap of U.S. UFO sighting counts by state × decade (1960s–2010s), color-encoded with a log scale
- **Plot 2:** Interactive strip plot of sighting duration by UFO shape, with a dropdown filter
